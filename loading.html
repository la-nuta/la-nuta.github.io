About loading component

Loading component with css animation.

Loading-component is designed for loading page elements such as images.

Demonstration time of loading component is defined in constanta "DELAY_BEFORE_FINISH" in loading-component.js. Demonstration of loading component lasts not less than 1500ms.

Maximum demonstration time of loading depends on duration of loading the particular element.

How to use loading component

Example of code to use loading component:

import {LoadingComponent} from "'path'/components/loading";

const loadingComponent = new LoadingComponent(wrapper, image);

// where:

// 'path' is the path to component's folder;

// wrapper is a wrapper for an image;

// wrapper should have width and height defined in pixels;

// wrapper should be positioned relative.
