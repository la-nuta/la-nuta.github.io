About modal-dialog component

Modal-dialog component creates dialogs with different content.

Phrases in the dialogs are switched by interaction with the button "Next".

Click on the button "Next" shows a new phrase.

Keyboard key "escape" skips the dialog. Keyboard keys "arrow right", "space" and "enter" also show a new phrase.

This component doesn't cover the whole page and doesn't prevent other events on the page.

The width of the dialog depends on the width of its wrapper.

How to use modal-dialog component

Content is defined in phrases.js, that stores the object in the variable "phrases".

Different options of dialogs are defined in the object with the structure:

{

dialogName1 or dialogsName1 : [

// Array with dialog phrases or Array of arrays with dialog phrases

],

dialogName... or dialogsName... : [

// Array with dialog phrases or Array of arrays with dialog phrases

],

dialogNameN or dialogsNameN : [

// Array with dialog phrases or Array of arrays with dialog phrases

]

}

So each property of the object in the variable "phrases" contains: the whole dialog (array) with different number of phrases (string) or the group of dialogs with common name (array of arrays of such dialogs).

If the property contains the group of dialogs the current dialog is chosen at random.

After showing the last phrase of the current dialog the modal-dialog disappears.

Example of code to use modal-dialog component:

import {ModalDialog} from "'path'/components/modal-dialog";

const newDialog = new ModalDialog(wrapper, dialogName/dialogsName);

// where:

// 'path' is the path to component's folder;

// wrapper is a wrapper for dialog;

// dialogName is a name of a dialog;

// dialogsName is a name of a group of dialogs.
